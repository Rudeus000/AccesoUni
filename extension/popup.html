Acciones

Comandos de voz

  • Perfil «Lector de pantalla»: conviene dejar encendida la opción de primera tecla o clic arriba para no buscar el botón. Atajo fijo: Alt + Mayús + V.
  • Otros perfiles: botón Activación por voz, Alt + Mayús + V, o primera tecla/clic si marcó la opción.
  • Funciona sobre una pestaña normal https: el navegador pide micrófono en esa página cuando empieza a escuchar.
  • Perfil lector de pantalla: activate, activar perfil ciegos, lector de pantalla u otras frases similares.
  • Reaplicar la configuración: activar acceso uni, activar versión, activar accesouni o activar modo de voz.
  • Leer pantalla o leer todo el texto (voz o Alt+Mayús+T). Describir imágenes para oír alt y nombres detectados (Alt+Mayús+I).
  • Para dejar de escuchar: pare, cancelar o basta.

Cuenta y servidor

Para guardar preferencias en la nube hace falta una sesión. Abra el inicio de sesión en el mismo API que usa la extensión, copie el token y péguelo aquí.
